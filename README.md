# Normix AI — Trade Compliance OS

**The intelligent operating system for African trade compliance.**

Normix AI is a high-performance, AI-driven platform designed to streamline cross-border trade operations for African businesses. By integrating advanced logistics tracking with automated regulatory compliance, Normix eliminates the friction of international trade in the African Continental Free Trade Area (AfCFTA) and beyond.

---

## 🚀 Vision

To empower African enterprises with digital infrastructure that makes cross-border commerce as simple as a local transaction. Normix AI acts as the "Protocol Layer" for trade, ensuring every shipment is classified correctly, tracked accurately, and compliant with regional regulations.

## 🛠 Features

### 1. AI-Powered Customs Classification
Leverages Gemini AI to interpret complex Harmonized System (HS) codes, ensuring accurate duty calculations and preventing costly delays at customs points.

### 2. Real-Time Shipment Orchestration
A unified dashboard for managing multi-modal shipments across the continent. Track logistics from origin to final destination with granular visibility.

### 3. Compliance Intelligence
Automated risk assessment and regulatory monitoring. Stay ahead of shifting trade policies and documentation requirements in over 54 African nations.

### 4. Zero-Trust Security
Secured by Firebase Enterprise and state-of-the-art authentication protocols, ensuring your sensitive trade data remains private and protected.

## 🏗 Tech Stack

- **Frontend:** Next.js 15 (App Router), TypeScript
- **Styling:** Tailwind CSS v4 (Mobile-First Precision)
- **Animations:** Motion (Framer Motion)
- **Backend/DB:** Firebase (Firestore, Authentication)
- **AI Engine:** Gemini 1.5 Pro via Google AI
- **UI Components:** Lucide Icons & Custom-Crafted Polish

## 🚦 Getting Started

### Prerequisites

- Node.js 20+
- A Google Cloud/Firebase Project

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd normix-ai
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file based on `.env.example` and populate your Firebase and Gemini credentials.

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```

Normix AI is not just a tracking tool; it is a **single source of truth** for trade across the 54 nations of Africa.

- **AfCFTA Readiness:** Native support for the African Continental Free Trade Area protocols, automatically calculating Rules of Origin (RoO) and tariff preferences.
- **Automated Customs Defense:** Generates pre-audited dossiers that reduce inspection times at ports like Mombasa, Lagos, and Durban by up to 60%.
- **Zero-Latency Compliance:** Real-time synchronization with regional trade gazettes, ensuring your HS Code classifications are always current.
